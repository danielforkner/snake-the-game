# snake-the-game

[LIVE DEMO](https://danielforkner.github.io/snake-the-game/)

Use WASD, ArrowKeys, and buttons(mobile) to control Snake.
Please email bugs and suggestions to daniel.forkner@gmail.com.

FEATURING:

- 3 bosses
- boss spells
- powerups

Tech:

- vanilla Javascript
- https://gifmaker.me/
- https://pixlr.com/
